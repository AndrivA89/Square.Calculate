Пример использования библиотеки:

Figure circle = new Circle(3);
Figure triangle = new Triangle(5, 3, 4);
Console.WriteLine("Площадь круга - " + circle.Area());
Console.WriteLine("Площадь треугольника - " + triangle.Area());
Console.WriteLine("Треугольник прямоугольный? - " + triangle.RightAngled());
